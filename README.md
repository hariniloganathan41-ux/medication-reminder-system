
# Medication Reminder System

## Overview

The Medication Reminder System is a web-based application developed using PHP, MySQL, HTML, and XAMPP. The system helps users manage their medication schedules by adding, viewing, updating, and deleting medicine reminders. It also provides an alarm notification to remind users to take medicines on time.

## Features

* Add new medication reminders
* View all scheduled medications
* Edit existing medication details
* Update reminder information
* Delete medication records
* Alarm notification support
* Simple and user-friendly interface

## Technologies Used

* HTML
* PHP
* MySQL
* JavaScript
* XAMPP

## Project Files

* index.html – Home page
* add.php – Add medication records
* view.php – View medication records
* edit.php – Edit medication information
* update.php – Update medication details
* delete.php – Delete medication records
* api.php – Backend processing
* alarm.mp3 – Reminder notification sound

## Database Setup

1. Install XAMPP.
2. Start Apache and MySQL.
3. Open phpMyAdmin.
4. Create a new database.
5. Import the provided SQL file.
6. Copy the project folder into the htdocs directory.
7. Open the project in a browser using localhost.

## How to Run

1. Start Apache and MySQL in XAMPP.
2. Open your browser.
3. Navigate to:
   http://localhost/Medication-Reminder-System
4. Use the application to manage medication reminders.
